关于名称、Logo

代码，直接突出主题；星冰乐，是我最喜欢喝的一种咖啡；代码星冰乐，表层寓意为一种自创的咖啡品种，另一种寓意是敲着代码喝着咖啡的一种上进、自在、安适的心境。
我认为人生最美好的事情就是喝喝咖啡，看看书，敲着代码，然后思考人生，眺望’远方‘～

Logo 采用了和咖啡同类的甜甜圈图片，寓意甜美的梦想；
两个小翅膀代表飞翔，寓意飞向梦想的翅膀；
整体来说，祝你可以飞向自己的梦想～

Code Frappuccino，就是代码星冰乐的英文，Frappuccino 是星冰乐，也有叫做“法布奇诺”。
